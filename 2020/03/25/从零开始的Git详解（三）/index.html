Git工作原理

工作区

工作区（Working Directory）就是在电脑里能看到的一个单独的文件夹，对应一个Project，如下图所示Test文件夹就是一个工作区。

版本库

Git版本库（Repostitory）是指工作区内隐藏的文件夹.git，这个文件夹存放了很多东西，其中最主要的是stage（或称为index）的暂存区，还有Git自动创建的第一个master分支，以及指向master的指针HEAD。

原理

  1. 第一步 git add，将工作区文件的修改内容添加到暂存区。

  2. 第二步git commit，将暂存区的内容都提交到当前的分支上。

    实操：
    可通过git status命令查看文件当前的状态
