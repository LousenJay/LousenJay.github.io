最近从头开始复习JavaSE，发现还是有很多不足之处，于是把复习中做过的比较经典的例题整理到这个笔记中，希望
自己能够完全理解、掌握这些经典基础题。

一. 经典基础题

1.1 打印等腰三角形

在控制台输出用*打印的等腰三角形

1.2 打印近似圆

在控制台用*打印近似圆

1.3 按字节截取字符串

注：一个英文字符相当于一个字节，一个中文字符相当于两个字节
例：输入字符串：”我们asd”,再输入（1,3）将返回”们as”
