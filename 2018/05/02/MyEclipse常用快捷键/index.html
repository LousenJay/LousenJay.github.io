MyEclipse常用快捷键

  • IDE
  • 2018-05-02

最近使用MyEclipse越来越感觉到快捷键的便利，可以大大提高编码的效率，以下整理了我在使用MyEclipse时较常使用的快捷键，并没有列出所有的快捷键，仅列出了我所认为的最实用的快捷键。

测试环境：
Windows 7 旗舰版
MyEclipse 2017 CI
jdk 1.8

最常使用

智能提示：Alt + /
单行注释/取消注释:Ctrl + /
多行注释：Ctrl + Shift + /
取消多行注释：Ctrl + Shift + \
删除当前行：Ctrl + D
快速修复：Ctrl + 1
前/后一个编辑页面：Alt + 左/右
新建文件：Ctrl + N

较常使用

移动当前行到上/下一行：Alt + 上/下
在当前的下一行插入换行：Shift + Enter
选择大一级的元素：Alt + Shift + 上
“Source”快捷键:Alt + Shift + S
添加try/catch块：Alt + Shift + Z
导入当前位置所需的包:Ctrl + Shift + Ｍ
最大化当前视图：Ctrl + M
保存所有文件:Ctrl + Shift + S
关闭所有文件：Ctrl + Shift + F4
快速显示当前类的继承结构：Ctrl + T
显示当前资源的属性：Alt + Enter
格式化代码：Ctrl + Shift + F
抽取方法：Alt + Shift + M
